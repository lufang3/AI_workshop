## 1. 상반기 성과

# 매출 목표 대비 112% 달성, 핵심 성과 3축에서 동시 성공
- 매출 목표 대비 112% 달성
- 신규 거래처 8곳 확보
- 업무 자동화로 월 40시간 절감

# 4대 전략 과제를 모두 완료해 운영 안정성을 확보했다
- 사내 문서 표준화 프로젝트 완료
- 고객 응대 매뉴얼 개정
- 분기별 정기 보고 체계 구축
- 협력사 평가 시스템 도입

## 2. 운영 진단

# 부서별 협업 모델이 안착되어 시너지 효과가 나타나고 있다
- 영업팀: 제안서 템플릿 공동 개발
- 재무팀: 월 정산 프로세스 간소화
- IT팀: 업무 데이터 대시보드 구축 지원

# 그러나 단순 반복 업무 비중이 여전히 전체 업무의 30%다
- 사무직 1인당 주 평균 12시간이 반복 업무에 소요됨

# 현장에서 발견된 6대 개선 과제
- 반복 업무의 수작업 비중이 여전히 높음
- 부서 간 데이터 공유가 지연되어 보고 지연 빈발
- 보고서 양식이 부서마다 제각각이라 통합 시 추가 작업 발생
- 회의 결과 추적 체계가 부재해 의사결정 추적이 어려움
- 외부 협력사와의 자료 송수신이 메일 위주로 비효율적
- 신입사원 온보딩이 부서별 임의 방식으로 진행됨

## 3. 하반기 전략

# 바이브 코딩으로 업무 도구 5종을 직접 제작한다
- 사무직 직원이 직접 만들어 직접 쓰는 자동화 도구를 표준으로 만든다

# 데이터·자동화·인재의 3축으로 디지털 업무 역량을 끌어올린다
- 데이터: 부서 공통 대시보드 운영
- 자동화: 반복 업무 도구 사내 확산
- 인재: 신규 입사자 디지털 온보딩 프로그램 운영

# 분기별 마일스톤으로 12개월 내 전사 적용을 완성한다
- 3분기 초: 시범 부서 3곳에 도구 5종 적용
- 3분기 말: 시범 결과 평가 및 도구 보완
- 4분기 초: 전 부서 확산 및 사용 교육 실시
- 4분기 말: 효과 측정 및 익년 KPI 확정

## 4. 예상 효과

# 단순 반복 업무 50% 이상 자동화로 연 4,800시간 절감을 목표한다
- 단순 반복 업무 50% 이상 자동화
- 정기 보고 작성 시간 70% 단축
- 연간 약 4,800시간 절감 (50명 기준)
- 인건비 환산 약 2.4억 원

# 정량 효과를 넘어 팀 전체의 디지털 업무 역량이 향상된다
- 직원 만족도 상승, 단순 반복 업무에서 해방된 시간을 고부가가치 업무에 재배치
